# game-ua-localization-tools

## Як витягнути TSV з TLK XML (для не технічних користувачів)

1. Відкрийте репозиторій у Codespaces.
2. Завантажте файл `GlobalTlk_tlk.xml` у папку `input/` (перетягніть файл у дерево проєкту).
3. Запустіть команду в терміналі:

   ```bash
   bash run_extract.sh
   ```

4. Завантажте `locale/uk.tsv` з дерева файлів (правий клік → Download).

### Важливо для перекладачів
- **Не змінюйте** поля `id`, `flags` і `source`.
- Заповнюйте **тільки** колонку `translation`.
- Для переносу рядка використовуйте літерал `\n` (не натискайте Enter всередині клітинки).

### Якщо ви редагуєте не врун а перекладаєте через GPT використате промпт для зберження структури файлу:

Ти перекладаєш TSV для локалізації гри (EN→UK). Я надсилаю блок рядків TSV.
Формат кожного рядка: 4 колонки, розділені TAB: id<TAB>flags<TAB>source<TAB>translation.

Твоє завдання:
1.	У кожному рядку не змінюй id, flags і source.
2.	Заповни/онови лише 4-ту колонку translation українським перекладом source.
3.	У translation заборонено дублювати англійський текст із source (ніяких “спочатку англійське, потім українське”).
4.	Усі технічні токени в source (наприклад %s, %d, {0}, \n, \t, <ip/...>, будь-які <...> теги, KEY_*, [[...]]) повинні з’явитися в translation без змін.
5.	Якщо source порожній — translation лишай порожнім.
6.	Поверни тільки TSV-блок у тому ж порядку рядків, з тією ж кількістю рядків, без пояснень, без нумерації, без таблиць, без зайвих пробілів, збережи TAB-розділювачі.
7.	Якщо рядок критично двозначний без контексту — обери найтиповіший варіант для ігор, але додай в кінці перекладу  [REVIEW].

Після відповіді я вставляю блок назад у файл через Ctrl-V, тому формат має бути ідентичним.

## Як вставити переклад назад у TLK XML

1. Переконайтесь, що `locale/uk.tsv` містить переклади у колонці `translation`.
2. Запустіть команду:

   ```bash
   bash run_apply.sh
   ```

3. Завантажте `output/GlobalTlk_tlk.uk.xml` з дерева файлів.

> **Попередження:** зберігайте всі службові токени у перекладі та не натискайте Enter у TSV. Для переносу рядків використовуйте `\n`.

### Демонстраційний файл
У папці `input/` є `demo.xml`, щоб протестувати роботу без ігрових файлів.
