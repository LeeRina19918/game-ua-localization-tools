# game-ua-localization-tools

## Як витягнути CSV з TLK XML (для не технічних користувачів)

1. Відкрийте репозиторій у Codespaces.
2. Завантажте файл `GlobalTlk_tlk.xml` у папку `input/` (перетягніть файл у дерево проєкту).
3. Запустіть команду в терміналі:

   ```bash
   bash run_extract.sh
   ```

4. Завантажте `locale/uk.csv` з дерева файлів (правий клік → Download).

> **Важливо:** CSV містить поле `id`, яке потрібне для повторної вставки у другому скрипті. Будь ласка, **не змінюйте** значення `id`.
